Getting Started

  1. Install Node.js, a javascript-based server platform.
  2. Make sure config/helper.json is present (settings).
  3. Open a Command Prompt in this folder and enter: > node helper.js
  4. Now configure your browser / application with proxy settings: server address: 127.0.0.1 server port: 80
  5. Its done. Time to test.
